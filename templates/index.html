LET'S STYLIZE YOUR PHOTO LIKE IT WAS MADE BY PICASSO

Neural Stylize your photo with us. This tool is an application of deep learning and neural style transfer. Pytorch and Scipy are at the heart of this project.
Get your own stylized photo by following these steps:

1. Select the content image

Upload the image you want our neural net to stylize.

2. Select a Style image

The style from this image will be extracted and applied to content image. Select one image from existing style images.

3. You get your stylized image

The model will paint your image with its artistic skills.

Before we get started:

This website is an extension of style transfer command line tool. Use the command line tool if you want to stylize large photos. The usage is entirely described in the README. The cli also has a feature for live webcam style transfer. For using live style transfer it is recommended to use GPU, i.e., cuda = 1. It works flawlessly on Nvidia GTX 1050.
